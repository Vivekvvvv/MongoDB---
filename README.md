# 简易电商平台 (MongoDB + Express)

这是一个基于 Node.js 和 MongoDB 的简易电商平台演示。

## 功能

-   **商品展示**: 首页网格化展示所有商品。
-   **商家后台**: 简单的表单用于发布新商品（名称、价格、描述、图片）。
-   **购物车**: 支持添加商品、查看购物车、计算总价。
-   **模拟结算**: 简单的订单提交功能。

## 安装与运行

1.  **安装依赖**:
    ```bash
    npm install
    ```

2.  **配置数据库**:
    确保 `.env` 文件中的 `MONGODB_URI` 指向正确的 MongoDB 地址。
    ```env
    MONGODB_URI=mongodb://127.0.0.1:27017/my_shop_db
    ```

3.  **启动服务器**:
    ```bash
    npm start
    ```

4.  **访问**:
    打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 使用说明

1.  **发布商品**: 在页面顶部的蓝色区域输入商品信息，点击"上架商品"。
2.  **购物**: 点击商品卡片上的"加入购物车"按钮。
3.  **结算**: 点击右上角的购物车图标，然后点击"去结算"。
